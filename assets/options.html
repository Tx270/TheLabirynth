How to play?

The point of the game is to finish 8 mazes as fast as possible!
Mazes are getting bigger and harder with each maze you finish.

To move you can use arrows and "WSAD", by using them at the same time you can do a double move.
To finish the maze you need to open a door by collecting a key.

You can use a bomb to blow up a wall. To use a bomb you need to turn to the wall
you want to blow up and you then press "C" or the SPACE key. You can only use one bomb per maze.

If you find any bugs feel free to open an issue at our github rep.
